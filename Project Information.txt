Prerequisites
-------------
# Install nodeJS in to the machine. Check it's version,
    node -v
# Install "Live Server" privided by RItwick Dey for VS Code.
  Modyfy the exten's settings by clicking the gread-icon. Search for "full reload" & checked the "Live Server Full Reload".
# Modify a tailwind config file using the cmd.
    npx tailwindcss init

Initial Setup & Start Watcher to Compile Automatically
-------------------------------------------------------
# Inside "tailwind.config.js" file's content section, define where the HTML file locates. Look for all HTML files inside the "build" dir.
    module.exports = {
        content: ['./build/*.html'],
        ...........,
    }
# Create "src/input.css" & include @tailwind base, components & utilities. It'll require to compile tailwind powered css file inside the "build" dir.
# Run the command to compile CSS by tailwind from the terminal where the "tailwind.config.js" locates.
    npm tailwindcss -i ./src/input.css -o ./build/css/style.css
# Link the "build/css/style.css" file inside the "build/index.html" file.
# Then run the cmd, so that any changes made inside the "/build/index.html" file, tailwind will re-compile the css file automatically.
[NB]: Make sure to locate the terminal where the "tailwind.config.js" file locates before executing the cmd.
    npx tailwindcss -i ./src/input.css -o ./build/css/style.css --watch
