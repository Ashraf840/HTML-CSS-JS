4 types (levels) of common CSS selector:
  • Universal Selector: Selects everything (HTML DOMs) on the page.
  • Element Selector: Selects all the specified elements of the page.
  • Class Selector: Selects based on the class of the HTML DOM element(s). Since class-selectors are more specific, they overrule element-selector.
  • ID Selector: Selects based on the ID of the specified HTML DOM element. They overrule class-selectors. It's not a goot practise to use IDs inside of CSS, since it's other valid use-cases.
